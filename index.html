Project Overview

This project aims to help beginners with their first contributions.

It supports both technical and non-technical contributions! It allows contributors to translate languages from english to spanish, french, russian, etc.

People who use it are beginners and those who may need a language translation in order to be able to understand what is going on.

Non-Code Contribution Opportunities

Some oportunities are to create a new issue, create pull request for existing issues or for changes ade for betterment of this repo and translation for others.

Contribution Guidelines

The guidelines are found in the contributing file.

If listed guidelines aren't being followed, you may be banned from the repository and your pr can be marked as invalid.

The onboarding process is beginner friendly as this is who this repository is mainly targeted for.

You have to be a beginner if you are making the contributions and only two contributions are allowed per person. Nothing really confused me about this because it is a repository meant for beginners.
