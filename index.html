Project Overview

This project aims to help beginners with their first contributions.

It supports both technical and non-technical contributions! It allows contributors to translate languages from english to spanish, french, russian, etc.

People who use it are beginners and those who may need a language translation in order to be able to understand what is going on.

Non-Code Contribution Opportunities

Some oportunities are to create a new issue, create pull request for existing issues or for changes ade for betterment of this repo.

You have to be a beginner if you are making the contributions and only two contributions are allowed per person.

If you are not a beginner the only way you are supposed to contribute to this repository is by guiding or solving doubts of the beginners
